# gestao_ferramentas

Django project scaffold created with `django-admin startproject`.

Quick start (PowerShell):

```powershell
Set-Location 'C:\Users\automacao\my-project\Controle_Ferramenta'
.\.venv\Scripts\Activate.ps1
python -m django --version
python manage.py runserver
```

Notes:
- Virtual environment was created at `.venv` and Django installed there.
- Database: SQLite (`db.sqlite3`) (ignored by `.gitignore`).
